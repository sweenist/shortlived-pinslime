# Short-lived Pin Slime

[![Deploy to Itch.io (HTML5)](https://github.com/sweenist/shortlived-pinslime/actions/workflows/deploy.yml/badge.svg?branch=main)](https://github.com/sweenist/shortlived-pinslime/actions/workflows/deploy.yml)

A quick and fun little game about a slime in a pinball maze of hell. Slimes have a short lifespan. Collect as many points as possible before the slime expires. Avoid walls to prevent premature death!

Templated from [typescript-game-engine-template](https://github.com/sweenist/typescript-game-engine-template)

## Build and run

- `npm i`
- `npm run dev`

Navigate to `http://localhost:5173`. Use spacebar, mouse or tap to work paddles. Click/tap options or use arrow keys and space to select
